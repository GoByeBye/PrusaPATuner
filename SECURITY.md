# Security policy

## Reporting a vulnerability

Please use GitHub's private vulnerability reporting feature for this repository when
it is available. If it is not enabled, contact the fork maintainer privately through
their GitHub profile instead of opening a public issue.

Include the affected version or commit, reproduction steps, impact, and any suggested
mitigation. Do not include real printer credentials, API keys, private network details,
or run captures containing sensitive data.

This experimental software controls network-connected hardware. Do not test a report
against a printer you do not own or have explicit permission to operate.
