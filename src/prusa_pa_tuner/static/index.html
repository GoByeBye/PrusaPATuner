Touch Probe — lateral sensitivity test

Step 1: characterise the sensor before building the probe. This pushes the nozzle tip sideways (±X / ±Y) into a rigid target and records the Z loadcell while it creeps in, to find out whether lateral contact registers a clean, repeatable force at all. The loadcell is an axial (Z) sensor; lateral force reaches it only as a bending moment, so this is an open question we have to measure.

Safe by design: the creep is short (the creep distance is a hard cap on how far we can push past contact) and slow, and the nozzle is cold by default — so even if nothing stops the move, the worst case is a gentle lean, not a crash. Set up: clamp a rigid object so the nozzle just touches it partway through the creep, then point the probe at it with the standoff + axis below. Contact should land after the first ~20 % of the creep (that early part is used as the no-contact baseline).

Touch Probe — results

Force vs. probe-axis position, every touch overlaid. A usable lateral probe needs a clean knee: force flat (no contact), then a sharp, monotonic rise — landing at the same position on every touch. Vertical dashed lines mark the detected contact point per touch; tight clustering = good repeatability.

Max Flow — results

Steady-state nozzle force vs flow rate. The fitted curve is the well-behaved power law F = a·Qᵇ + c; markers show where force breaks upward off it (melting limit), where per-level noise rises (instability onset), and where force collapses (extruder skipping). Recommended max = earliest soft breakdown, derated ~15%. The amber curve (right axis) is the per-step force σ (noise) — its axis is scaled to the quiet low-flow steps, so the post-skip σ explosion clips off the top rather than flattening the rest; hover for true values.
